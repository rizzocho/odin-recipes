Simple Hamburger Stroganoff

Description

Stroganoff is rich and creamy, this version uses ground beef. It can be made quickly and easily.

Ingredients

  • ground beef
  • noodles
  • sauce
  • green stuff (optional)

Steps

  1. Brown the meat, set aside
  2. Boil the noodles, set aside
  3. Heat the sauce
  4. Combine all ingredients, and enjoy
